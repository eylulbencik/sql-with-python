# SQLite3 with Python: Fundamentals & Practice

This script demonstrates how to interact with an SQLite database using Python's built-in `sqlite3` module. It includes a complete workflow from database creation to executing complex queries safely.

## Key Features:
* **Database Setup:** Creating a database (`students.db`) and handling existing files using the `os` module.
* **Table Creation:** Defining `Students` and `Courses` tables with properties like `PRIMARY KEY`, `NOT NULL`, and `UNIQUE`.
* **CRUD Operations:** Inserting multiple records using `executemany()`, as well as updating and deleting data safely.
* **Data Retrieval:** Utilizing `SELECT`, `WHERE`, `LIKE`, `ORDER BY`, and `LIMIT` clauses to filter data.
* **Aggregate Functions:** Performing calculations using `COUNT`, `AVG`, `MAX`, `MIN`, and `GROUP BY`.
* **Practice Queries:** A set of beginner-to-intermediate SQL questions and their practical solutions.
